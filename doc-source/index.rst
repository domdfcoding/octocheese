#############
OctoCheese
#############

.. start short_desc

.. documentation-summary::
	:meta:

.. end short_desc

This is a GitHub action, Python package and command line script that copies distribution files from PyPI ("The Cheese Shop") and adds them to the relevant tag in GitHub Releases.
